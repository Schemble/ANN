In order to run the demo jupyter notebook file 'how_ipynb.ipynb'
yoy need to do the follwoing:

1. Install Anaconda and Tensorflow + Keras libraries. See the
attached pdf-file (preparation-computer-exercises.pdf) 
for more instructions.

2. Start the jupyter notebook from the Anaconda Prompt (windows)
or from a terminal (linux or macOS), i.e.

> jupyter notebook

3. Find the folder where you save the two files
'how_ipynb.ipynb'
'auto.png'

4. Click on the file 'how_ipynb.ipynb'
